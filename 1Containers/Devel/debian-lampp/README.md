Este projeto contém dois CRUds, um com mariadb e outro com postgresql
Os CRUDs usam PHP, PDO, paginação e bootstrap.

Copie

cp sample.env .env

Abra o .env e altere a gosto a versão do PHP

Então execute

docker-compose up -d

Ao final

docker images

docker run -it nomeimagem

Executar:
start.sh

ip a

Abra no desktop

http://172.17.0.2
